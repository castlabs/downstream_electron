# Redux Integration

Integration support `npm` and `yarn` managers.

### Redux

Directory `integrations/redux` contains a full example of integration **downstream_electron**
with React & Redux frameworks.

#### Development

1. `npm install`
2. `npm run dev`

or

1. `yarn install`
2. `yarn dev`

#### Release

1. `npm install`
2. `npm run dist`

or

1. `yarn install`
2. `yarn dist`
