castLabs Electron Release for Content Security

To simplify the use of Widevine DRM and allow protected playback of offline content within Electron CastLabs has created a fork with support for Widevine CDM installation, Verified Media Path (VMP), and protected storage of offline licenses. Such a release is installed by the default npm package scripts.

More information is available here:

https://github.com/castlabs/electron-releases
